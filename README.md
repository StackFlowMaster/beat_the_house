# Beat The House

Beat The House is a platform where users can build/test strategies for casino table games.

## Installation

Use the package manager npm to install dependencies

```bash
npm install
```

## Usage

```bash
#/beat_the_house/backend
rails s

#/beat_the_house/frontend
open index.html
```
